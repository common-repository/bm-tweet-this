A simple method for adding Twitter Buttons to your WordPress theme

== Description ==

Adds Twitter's new Tweet button to your posts. Also includes a control panel for setting the buttons options. Will soon include other Twitter based features to improve your WordPress & Twitter usage.

== Installation ==
1. Upload `bm-tweet-this` directory to the `/wp-content/plugins/` directory on your website
1. Activate the plugin through the plugins men in WordPress
1. Edit the options in 'Options' -> 'Twitter Button' to taste
